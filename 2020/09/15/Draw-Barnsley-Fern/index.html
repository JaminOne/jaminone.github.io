Draw New Zealand's National Identity Fern
Post：2020-09-15 15:59:02 Tags：/ Python / Matplotlib / Visit：

The silver fern is a symbol of New Zealand’s national identity. To Maori, the shape of the frond represents strength, resistance, and enduring power. To Pakeha(New Zealand of non-Maori descent), the fern symbolised sense of attachment of their homeland. Today we will use Python Numpy and Matplotlib libraries.

The fern leaf is based on affine transformations, and the formula for one transformation can be found the wiki page

The main ideas behind the project:

  • Create four functions to capture the stem and leaf in different shapes.
  • Create a random generator with probability by using numpy random.choice.
  • Comparing to list, the np array can draw better graph, but it requires to shift the point and mark the value 1 on the array.
